# 🧮 Calculator

> A clean, minimal, and fully functional calculator built for the web — no bloat, just math.

**🔗 Live Demo → [calculator-by-dpnshuu.vercel.app](https://calculator-by-dpnshuu.vercel.app/)**

---

## ✨ Features

- ➕ Basic arithmetic — addition, subtraction, multiplication, division
- 💯 Percentage calculations (`%`)
- ➕➖ Toggle positive/negative values (`+/-`)
- 🔢 Decimal point support
- ⌫ Backspace to delete last digit
- 🔄 All Clear (`AC`) to reset instantly
- 📱 Responsive — works on all screen sizes

## 🚀 Run Locally

No build tools or dependencies needed — just open the file!

```bash
# Clone the repo
git clone https://github.com/Dpnxhuu/Calculator.git

# Navigate into the folder
cd Calculator

# Open in browser
open index.html
```

Or simply double-click `index.html` in your file explorer.
